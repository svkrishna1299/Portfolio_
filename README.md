# Raj Gopala Krishna S.V — Portfolio

Personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## 🚀 Deploying to GitHub Pages

1. **Create a repository** on GitHub named `your-username.github.io`  
   (or any repo name — GitHub Pages works for both)

2. **Upload files** — push all three files to the repo root:
   - `index.html`
   - `style.css`
   - `main.js`
   - `Resume_120326.pdf` ← place your CV PDF here too

3. **Enable GitHub Pages**:
   - Go to `Settings → Pages`
   - Source: `Deploy from a branch`
   - Branch: `main` (or `master`), folder: `/ (root)`
   - Click **Save**

4. Your site will be live at `https://your-username.github.io` within a minute.

## 📁 File structure

```
/
├── index.html        ← main page
├── style.css         ← all styles
├── main.js           ← animations & interactions
├── Resume_120326.pdf ← your CV (linked in the site)
└── README.md
```
